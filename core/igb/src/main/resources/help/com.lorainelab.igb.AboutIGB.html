About Integrated Genome Browser

IGB (pronounced ig-bee) is a fast, flexible, desktop genome browse
first developed at Affymetrix for tiling arrays.
IGB is now open source software supported by grants and donations.
To find out more, visit BioViz.org.

If you use IGB in your research, please cite
Nicol JW, Helt GA, Blanchard SG Jr, Raja A, Loraine AE.
The Integrated Genome
Browser: free software for distribution and exploration of
genome-scale datasets. Bioinformatics. 2009 25(20):2730-1.
