GenoViz SDK NeoMap & NeoSeq Interaction Demo

If you see this text, your browser apparently doesn't support Java!

This demo shows how one widget can be "shadowed" on another. The two widgets have been set up so the range of sequence shown in the NeoSeq (above) is represented in the NeoMap (framed in a floating window) by a red shadow along the axis. Any changes in this range are propagated to the shadow. So as you scroll the NeoSeq, you can watch the red outline move along the axis of the NeoMap.

You will need to close the floating window in order to leave this page.
