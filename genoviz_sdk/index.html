Affymetrix GenoViz Software Development Kit

Welcome to Affymetrix GenoViz Software Development Kit

The Affymetrix GenoViz SDK is a set of Java-based components for building sophisticated bioinformatics data-visualization applications.

The Affymetrix GenoViz SDK consists of a number of "widgets" that can be used to visualize many types of bioinformatics data. The following widgets are available in GenoViz:

  • NeoMap for annotated linear maps (physical, genetic, sequence, etc.)
  • TieredNeoMap for annotated linear maps with several groups of data.
  • NeoAssembler for sequence assemblies and alignments
  • NeoSeq for sequences at residue-letter resolution
  • NeoQualler for sequencing quality scores
  • NeoTracer for DNA sequencing traces
Refer to the Tutorial for more information on these widgets.
