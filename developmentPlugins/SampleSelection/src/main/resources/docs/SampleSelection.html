Plugin Name: SampleSelection

Version: 1.0.0

Date: 2011-05-23

Author: Genentech

Overview:    tab to allow the user to select sample data.
  
Usage:The SampleSelection plugin is an implementation of the IGBTabPanel extension point.
When loaded, it will display a new tab "Sample Data" in the tab panels on the sides/bottom of the screen.
This is a general use plugin used by other plugins to allow the user to select sample data
to pass to the other plugin.
The other plugin will fill in the Sample Data tab with the sample data, if any. The Samples
are displayed in a single column table vertically with a Scroll bar if needed. The Variable
dropdown contains the names of all the variables for the samples. The user can select the variable
to bring up all the samples for that variable, and then select the samples (multiple selection allowed)
for the variable. Then the next variable can be selected in the dropdown, and the process repeated.
This way, multiple combinations of Variable/Sample can be selected. Then when the Apply button is
pressed, the selected variable/sample combinations are passed back to the other plugin. The
selected combinations can be modified and the Apply button pressed when needed.
