Plugin Name: TallyHandler

Version: 1.0.0

(Designed to work with IGB6.7 and up)

Date: 2011-11-15

Author: Genentech

Overview:    plugin to allow the user to load Tally files.
  
Usage:This plugin is used to allow the user to load Tally files in IGB. It is an implementation of the FileTypeHandler extension point. Tally files are created by the bam_tally program (using options -P -B 0), and contain mismatch pileup information. The display is identical to the MisMatch Pileup view mode. The Tally files contain the sequence reference. The files must have the file extension .tally. The plugin will use a tabix index if available.
