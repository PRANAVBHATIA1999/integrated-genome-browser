Plugin Name: OverlapAnnotationOperator

Version: 0.0.9

(Designed to work with IGB6.7 and up)

Date: 2011-08-26

Author: Genentech

Overview:    plugin to allow the user to find symmetries that overlap on two selected annotation tracks.
  
Usage:The OverlapAnnotationOperator plugin creates a new track containing all the annotations from
the two selected tracks where the annotations on one overlap an annotation on the other. It is an implementation of the AnnotationOperator
extension point. When loaded, the Track Operations popup menu tab will have an option "Subset by overlaps", and if selected, it will display the new track.
