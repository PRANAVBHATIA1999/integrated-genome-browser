Plugin Name: GeometricMean

Version: 1.1.0

Date: 2011-05-23

Author: Genentech

Overview:    operation to compute Geometric Mean on selected graph tracks.
  
Usage:The GeometricMean plugin computes the geometric mean of selected graph tracks.
It is an implementation of the GraphOperator extension point. When loaded, the Operation dropdown on the
Graph Adjuster tab will have an option "Geometric Mean", and if selected, and the Go button pressed,
it will create a new graph track with the value at each
position equal to the Geometric Mean of all the selected graph tracks at that position.
At least two graph tracks must be selected.
