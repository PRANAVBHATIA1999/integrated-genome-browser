GenoViz

Tools for Genomics Data Visualization

The GenoViz project includes three components:

  • GenoViz SDK (software development kit) is a library of re-usable components for genomics data visualization.
  • Genometry is a unified data model to represent relationships between biological sequences.
  • Integrated Genome Browser (IGB, pronounced "ig-bee") is an application built upon the GenoViz SDK and Genometry for visualization and exploration of genomes and corresponding annotations from multiple data sources. IGB is also a client for the DAS (Distributed Annotation System) protocol.
