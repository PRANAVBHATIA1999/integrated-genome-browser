Plugin Name: MergeAnnotationOperator

Version: 1.0.0

(Designed to work with IGB6.7 and up)

Date: 2011-08-26

Author: Genentech

Overview:    Annotation Operator that merges selected annotation tracks.
  
Usage:The MergeAnnotationOperator plugin creates a new track containing all the annotations from
the selected tracks. It is an implementation of the AnnotationOperator extension point. When loaded, the Track Operations popup menu tab will have an option "merge", and if selected, it will display the new merged track.
