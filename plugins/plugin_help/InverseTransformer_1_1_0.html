Plugin Name: InverseTransformer

Version: 1.1.0

Date: 2011-03-01

Author: Genentech

Overview:    Graph Transformer that takes the inverse of the input - 1/x .
  
Usage:The InverseTransformer plugin computes the inverse of selected graph tracks.
It is an implementation of the FloatTransformer extension point. When loaded, the Transform dropdown on the
Graph Adjuster tab will have an option "Inverse", and if selected, and the Go button pressed, it will create a
new graph track for each selected track with the value at each position equal to the inverse of the selected graph
track at that position.
