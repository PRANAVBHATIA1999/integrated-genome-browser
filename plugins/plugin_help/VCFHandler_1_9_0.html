Plugin Name: VCFHandler

Version: 1.9.0

(Designed to work with IGB6.7 and up)

Date: 2011-05-23

Author: Genentech

Overview:    plugin to allow the user to load VCF files.
  
Usage:This plugin is used to allow the user to load VCF files in IGB. It is an implementation of the FileTypeHandler extension point.
VCF files are text annotation files described in VCF 4.0 format and VCF 4.1 format.
They must have the file extension .vcf, and can be index with a tabix file .vcf.tbi. VCF files without tabix can be used,
but they will be much slower to load. They can also be compressed with gzip for a .vcf.gz extension.
The plugin requires the SampleSelection plugin to display and select variants as separate tracks,
which should be loaded automatically when the VCFHandler plugin is loaded.
