Plugin Name: CodonDisplay

Version: 1.0.0

Date: 2011-06-03

Author: Genentech

Overview:    display the codons for a selected track.
  
Usage:The CodonDisplay plugin is an implementation of the GlyphProcessor extension point.
This will display the Amino Acid codes over any annotation glyph that
contains base sequences (bam, sam). The codes will be displayed starting with the start
codon (Methionine) to the end codon. An error message is displayed over the annotation
glyph for any problems. A codon display menu item is added to the view menu with three options:
Note - currently only file types .brs and .bgn are supported.
  • 3 letter - to display the 3 letter amino acid codes
  • 1 letter - to display the 1 letter amino acid codes
  • hide - to hide the display - the annotation glyphs will appear normally
