Plugin Name: PowerTransformer

Version: 1.1.0

Date: 2011-03-01

Author: Genentech

Overview:    Graph Transformer that takes the input to a given power - x^a .
  
Usage:The PowerTransformer plugin computes the power of selected graph tracks to a given exponent.
It is an implementation of the FloatTransformer extension point. When loaded, the Transform dropdown on the
Graph Adjuster tab will have an option "Power", and if selected, it will display an exponent field.
If the exponent field has a valid number entered, and the Go button pressed, it will create a new graph track for each selected
track with the value at each position equal to the power of the selected graph track at that position to the entered exponent.
The Transform dropdown will also have an option "Sqrt" which is the Power transform with a fixed exponent of 0.5
(the exponent field will not be displayed) as a convenience.
