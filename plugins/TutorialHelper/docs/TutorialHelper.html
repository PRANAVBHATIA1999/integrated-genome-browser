Plugin Name: TutorialHelper

Version: 0.0.9

(Designed to work with IGB6.7 and up)

Date: 2011-09-28

Author: Genentech

Overview:    plugin to for tutorial developers to see widget ids
  
Usage:The TutorialHelper plugin is an implementation of the JRPWidgetDecorator extension point.
The TutorialHelper plugin is only for tutorial writers. It displays the Record/Playback id of any widget as a tooltip.
The tooltip replaces the original tooltip of the widget.
  

If you feel like the plugin has a defect, you can fill out a bug report in SourceForge Genoviz.
