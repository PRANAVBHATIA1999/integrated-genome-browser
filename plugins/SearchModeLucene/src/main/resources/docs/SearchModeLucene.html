Plugin Name: SearchModeLucene

Version: 0.0.9

(Designed to work with IGB7.0 and up)

Date: 2011-12-16

Author: Genentech

Overview:    a search that uses Apache Lucene indexing.
  
Usage:The SearchModeLucene plugin is an implementation of the ISearchMode extension point using Apache Lucene.
The search depends on the lucene index for each file, created by the LuceneIndexing project.
The user may enter a search term using the Lucene syntax, such as "seq:chr3 AND name:ADAR".
See http://lucene.apache.org/java/3_5_0/queryparsersyntax.html for details.
