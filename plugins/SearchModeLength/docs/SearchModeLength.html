Plugin Name: SearchModeLength

Version: 1.9.0

(Designed to work with IGB7.0 and up)

Date: 2011-07-08

Author: Genentech

Overview:    search mode to allow searching on length of annotation.
  
Usage:The SearchModeLength plugin is an implementation of the SearchMode extension point.
The search depends on the length of the gene, the user enters an operator, either
>, <, or =, then the length amount, no space, in the for box. The search will return
all genes that have length less than, equal to, or greater than the entered length.
