Track Defaults

This panel allows you to change default options for specific file types so when they are loaded they will be initialized with these values. Changes here will have no effect on files which are already loaded, but apply only to files loaded afterwards (Note: IGB will remember the settings of all previously loaded files, so changes here will only apply to files which have not previously been loaded). Properties specified by a server will also take precedence over these defaults, but if a server has not specified an attribute this panel will be referenced.

Adding a File Type

To add a file type to the table click the add button and select the desired file type from the list. To change attributes associated with this file type, select its row in the table.

Removing a File Type

To remove a file type from the table select the row and click the remove button. Removing the entry in this table will reset the values back to their defaults.
